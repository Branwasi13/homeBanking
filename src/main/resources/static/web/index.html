You don’t need to apply a location filter when searching for a job, but it will help narrow your search, since the more criteria you add, the less results you get. If you choose to add a location filter, it can be searched by entering city, state, and/or country in the Location Field. After you do this, the radius value will automatically apply a 25-mile radius, which can be adjusted to a minimum of 0 or a maximum of 100 miles by sliding the bar. You can then enter a new location by typing it into the text box, adjusting the radius (if needed) and then clicking Search. If you want to clear the location entirely, delete it from the text box, clear the radius, and then click Search.
